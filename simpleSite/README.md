# Simple Site

A React TypeScript application with authentication UI components built with Vite and Tailwind CSS.

## Features

- Login forms with different layouts
- Registration form
- Two-factor authentication components
- Dashboard interface
- Dark theme design
- Responsive grid layout

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Components

- `LoginForm` - Standard login form
- `SimpleLogin` - Alternative login layout
- `RegisterForm` - User registration form
- `TwoFactorAuth` - 2FA verification (registration flow)
- `TwoFactorLogin` - 2FA verification (login flow)
- `Dashboard` - Main dashboard interface

All components are purely presentational with no state management or navigation functionality.
